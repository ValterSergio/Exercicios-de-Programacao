Titulos e parágrafos

Quando precisamos usar titulos, subtitulos, ou outros niveis de texto temos as tags h1, h2, h3, h4, h5 e h6 cada número seguido da letra H minuscula possui um tamanho diferente de texto.
  Para criar parágrafos podemos usar a tag p ela permite escrever textos usando várias linhas de texto.
  Quando escrevemos textos sem tags. O texto não gera uma quebra de linha automaticamente para começar outra, nesse caso usamos a tag BR.
  Nesse caso devemos estar atentos ao uso correto:
  Inicio do texto: Na primeira linha antes do texto
Exemplo:
     <br> um texto qualquer
  Continuação de linha: No final da segunda linha
Exemplo:
     um texto qualquer <br>
  Final da terceira linha: Igual ao anterior
Exemplo:
     um texto qualquer <br>
